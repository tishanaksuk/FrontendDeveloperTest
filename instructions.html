Concepts included in this test

  • Manipulating Form Submission and Validation via JS
  • Cookies (setting/getting)
  • HTML5 Input Attributes
  • Graceful Degradation/Modernizr
  • Styling forms
  • Adding CSS/JS/jQuery to the header
  • Setup Bower to add libraries to the page
  • Setup Grunt/Gulp to compile JS and SASS

Instructions

Setup

1. Add jQuery Library
2. Add External Stylesheet
3. Add External JS file to contain all JavaScript

Design

1. Style Form: Remove list bullets, align fields into 2 columns, style input width according to their expected lengths.
2. Test that design is consistent in IE7+, Chrome, Firefox, Safari

JavaScript Development

On Page Load

1. Hide company field and setup event on commercial radio input to show company field when this radio button is selected.
2. Add HTML5 default placeholder attribute to add default values. Use modernizr for feature detection and offer backup JavaScript for browsers that dont support HTML5 input attributes.
3. Phone Input - Only allow numbers to be entered.
4. Zip Code Input - Only allow numbers and maximum of 5 digits.

On Form Submission

1. Setup regular expression on email address to validate that its a real email address, throw error if not match.
2. Compare the values in Email Address and Confirm Email Address and make sure they match on submit. If not, display an error message. Disable paste function on confirm email address field.
3. If State value is equal to NY, CA, or MA - show a message on submit saying that we are unable to ship wine to those states and hide submit button. If different state is chosen, onblur - check against eligible states and if can ship to then re-enable submit button.
4. Setup regular expression on phone number to validate that its a real US phone number, throw error if not match.
5. Setup HTML5 require attribute on the following fields: email, first name, last name, address 1, city, state, zip, phone. Use modernizr for feature detection and offer backup JavaScript for browsers that dont support HTML5 input attributes. Highlight fields that are missing values.
